# MultibotUtility - several bots to make your life with messengers easier

## GroupsBot - creates telegram groups to ping a group of persons

## usage

```
<@user1>: can any @admins answer to me?
<@multibot>: @user2, @user3, @user4
<@user2>: answering
```
